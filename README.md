# ChemPlasKin
0D sovler for unified gas-plasma kinetics simulations

## !!! Note !!! May 14 2024

The ChemPlasKin source code will be made publicly available upon the publication of its [paper](https://arxiv.org/abs/2405.04224) in a journal (expected late 2024). If you are interested in using ChemPlasKin in your research sooner, please contact us to discuss the possibility of accessing a precompiled version under specific conditions. We welcome collaboration opportunities, especially with experimentalists in gas-plasma kinetics. For inquiries, please contact [Professor Hong G. Im](https://www.kaust.edu.sa/en/study/faculty/hong-im).
